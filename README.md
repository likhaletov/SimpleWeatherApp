# SimpleWeatherApp (iOS 13+)

## Основной функционал
0. Поиск локаций реализован через MapKit Completer.
1. Отображение текущей локации пользователя.
2. Отображение погоды для списка локаций.
3. Прогноз погоды на 7 дней для городов из списка локаций.
4. Данные на устройстве хранятся в JSON-е.
5. API http://openweathermap.org
